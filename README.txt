Title: Real Time Bus Tracker;
Description Of Project:  
This project shows a bus moving between MIT and Harvard. 

How to Run:
To run the script download the repository and put the HTML file over your browser tab. You will need to update the mapboxgl.accessToken to one from your own MapBox account.  These are free up to a certain use.

Possible Future Imporvements:
Utalizing a real time tracker to follow the buses actual movements, vs just moving along the stops.
